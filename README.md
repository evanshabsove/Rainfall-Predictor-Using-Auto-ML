# 🌧️ Rain Predictor - Streamlit App

A decision support tool for event organizers to predict next-day rainfall based on current weather conditions.

## Features

- Real-time weather data integration via WeatherAPI.com
- Machine learning prediction using TPOT AutoML-optimized model
- Risk categorization (Low/Moderate/High)
- Event planning recommendations
- Clean, user-friendly interface

## Setup

1. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Get a free API key:**
   - Sign up at [WeatherAPI.com](https://www.weatherapi.com/)
   - Free tier includes 1 million calls/month

3. **Run the app:**
   ```bash
   streamlit run app.py
   ```

4. **First time running?**
   - Make sure `rain_predictor_model.pkl` and `preprocessing_info.pkl` exist in this directory
   - These are generated by running the training notebook in assignment_2

## Usage

1. Enter your WeatherAPI.com API key in the sidebar
2. Type in a location (city name or coordinates)
3. Click "Predict Rain Tomorrow"
4. View the risk assessment and recommendation

## Model Details

- **Trained on:** Australian weather station data
- **Optimization goal:** High recall (minimize missed rain events)
- **AutoML:** TPOT genetic programming
- **Key metrics:** Recall-optimized for event planning

## Notes

- Predictions are for the next day only
- Model prioritizes avoiding false negatives (missed rain predictions)
- Best suited for locations with similar climate patterns to training data
